# 基于科大讯飞语音合成接口实现配音

###### 参考项目[iflytek-tts](https://github.com/musistudio/iflytek-tts) 从该项目中改进而来的。
#### 使用方法
```python
iflytek = Iflytek()
iflytek.tts(text, vid) # text为需要合成的文本，vid为需要合成的音色
```
